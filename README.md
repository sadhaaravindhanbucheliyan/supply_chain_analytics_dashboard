**Supply Chain Analytics Dashboard**

## 📊 Overview
This project provides an interactive dashboard built in **Power BI** to visualize and analyze supply chain performance. By leveraging the provided dataset, the dashboard highlights key metrics, operational bottlenecks, and trends to support data-driven decision-making in supply chain management.

## 🚀 Features
- **Data-Driven Insights**: Analyzes key performance indicators (KPIs) related to supply chain efficiency.
- **Interactive Visualizations**: Users can filter and drill down into the data using the Power BI interface.
- **Comprehensive Analysis**: Covers various aspects of the supply chain lifecycle (as showcased in the provided screenshots).

## 📂 Repository Contents
- `supply_chain.pbix`: The source Power BI file containing the data model, measures, and dashboard design.
- `supply_chain_data.csv`: The underlying raw dataset used for the analysis.
- `supply chain page 1.png` / `supply chain page 2.png`: Visual previews of the dashboard layout and insights.

## 🛠 Tools Used
- **Power BI**: Used for data modeling, DAX calculations, and report visualization.
- **Microsoft Excel/CSV**: Used as the primary data source.

## 📈 How to Use
1. **Clone the repository** to your local machine.
2. **Open the `supply_chain.pbix` file** using Power BI Desktop.
3. If the data source path needs updating, go to **Transform Data > Data Source Settings** and re-link the `supply_chain_data.csv` file from your local directory.
4. Explore the pages to understand the supply chain metrics.
